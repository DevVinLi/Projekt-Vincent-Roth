# IB Score AG Website Dokumentation

## Beschreibung

Dieses Projekt ist eine Website für die IB Score AG, ein Inkassounternehmen. Die Website zeichnet sich durch ein modernes und responsives Design aus, wobei der Schwerpunkt auf klaren Handlungsaufforderungen (CTAs), reibungsloser Navigation und einfachen Animationen liegt. Sie unterstützt sowohl die englische als auch die deutsche Sprache.

## Projektstruktur

Die Projektdateien sind in einer klaren Ordnerstruktur organisiert:

```
Projekt Test/
├── en/               # Seiten in englischer Sprache
│   ├── index.html
│   ├── about.html
│   ├── services.html
│   ├── contact.html
│   └── media.html
├── de/               # Seiten in deutscher Sprache
│   ├── index.de.html
│   ├── about.de.html
│   ├── services.de.html
│   ├── contact.de.html
│   └── media.de.html
├── images/           # Platzhalter für Bilder (Logo, etc.)
├── script.js         # JavaScript für reibungsloses Scrollen, Animationen und Umschaltung des mobilen Menüs
├── style.css         # Cascading Style Sheets für die Gestaltung
└── README.md         # Dokumentationsdatei des Projekts
```

*   **en/**: Enthält die HTML-Dateien für die englischsprachige Version der Website.
*   **de/**: Enthält die HTML-Dateien für die deutschsprachige Version der Website.
*   **images/**: Ordner für Bilder, die auf der Website verwendet werden.
*   **script.js**: Enthält den JavaScript-Code zur Verbesserung der Benutzererfahrung und für interaktive Elemente.
*   **style.css**: Enthält alle CSS-Regeln zur visuellen Gestaltung der Website.
*   **README.md**: Diese Datei, die als Hauptdokumentation dient.

## Technologien

Das Projekt basiert auf den folgenden Web-Technologien:

*   **HTML5**: Für die Struktur der Webseiten.
*   **CSS3**: Für das Styling und Layout, einschliesslich Responsive Design.
*   **JavaScript**: Für interaktive Elemente und Animationen (Smooth Scrolling, Scroll Reveal).

## Wie man das Projekt ausführt

Da es sich um eine statische Website handelt, gibt es verschiedene Möglichkeiten, sie anzuzeigen:

1.  **Öffnen im Browser**: Sie können einfach jede der `.html`-Dateien in den Ordnern `en` oder `de` direkt mit Ihrem Webbrowser öffnen.

2.  **Lokaler Webserver**: Für eine bessere Emulation einer Live-Umgebung wird empfohlen, die Dateien über einen lokalen Webserver bereitzustellen. Navigieren Sie im Terminal zum Stammverzeichnis des Projekts (`Projekt Test/`) und führen Sie einen Befehl wie den folgenden aus:

    ```bash
    python3 -m http.server
    ```

    Öffnen Sie anschliessend Ihren Webbrowser und gehen Sie zu `http://localhost:8000` (oder der von Ihrem Server bereitgestellten Adresse).
